# Network_exam


### TCP 소켓 프로그래밍
+ 서버용 TCP 소켓 프로그래밍 순서
  + 1)) 서버 포트번호 지정
  + 2)) 서버용 소켓 객체 생성
  + 3)) 클라이언트의 요청 대기
  + 4)) 클라이언트의 요청 수락 후 클라이언트에 대한 소켓 객체 생성
  + 5)) 연결된 클라이언트와 입출력 스트림 생성
  + 6)) 보조 스트림으로 성능 개선
  + 7)) 스트림을 통해 읽고 쓰기
  + 8)) 통신종료
  
+ 클라이언트용 TCP 소켓 프로그래밍 순서
  + 1)) 서버의 IP주소와 포트번호를 매개변수로 하는 클라이언트용 소켓 객체 생성
  + 2)) 연결된 서버와 입출력 스트림 생성
  + 3)) 보조 스트림으로 성능 개선
  + 4)) 스트림을 통해 읽고 쓰기
  + 5)) 통신종료
